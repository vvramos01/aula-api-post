Trabalho de API REST desenvolvido em Node.js com Express.

Endpoints implementados:
- GET /
- GET /info
- POST /usuarios
- POST /login
- GET /api/me
- GET /api/data
- GET /api/random

Todos testados no Postman.
Collection incluída no repositório.
